# Hwang Youngyi Portfolio Website

This is a static portfolio site made for GitHub Pages. No build tools are required.

## Preview locally

Open `index.html` in a browser, or use any local static-file server.

## Publish with GitHub Pages

1. Create a new public repository on GitHub.
2. Upload everything inside this `portfolio-site` folder to the repository root.
3. Open **Settings → Pages** in the repository.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`, then click **Save**.
6. GitHub will show the published website address after deployment finishes.

## Before publishing

- Replace `hello@example.com` in `index.html` with the real contact email.
- Update the year or introduction text if needed.
- The site uses Google Fonts; replace the font link if fully offline hosting is required.
